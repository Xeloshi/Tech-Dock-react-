# ТЕСТОВЫЙ ОНЛАЙН МАГАЗИН ПРИ ИСПОЛЬЗОВАНИИ REACT
Тестовый проект в котором есть элементы онлайн магазина с рабочей корзиной и каталогом товаров, базой данных из которой берутся описания и значения товаров а так же немного элементов создания теста с некоторыми рабочими функциями (но это находится в разработке)
